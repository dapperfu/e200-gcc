It is recommended to have about 10 Gb of free space

To build full package (Linux and Windows versions):
1. Set correct path REPODIR in build_gnu/build.env-hostname as REPODIR=/path/to/this/readme.txt
2. Rename build.env-hostname with correct name of your machine
3. run ./build.sh ELvle

If you need just Linux version - adjust dLSTSTEPS config in build_gnu/f-jobs/ELvle
